# FinanceApp - Frontend

Built with **Vue.js 3** (Composition API), **Vite**, and **TailwindCSS**.

## Features
- **Authentication**: Login/Register (JWT stored in LocalStorage).
- **Dashboard**: Summary cards (Income/Expense/Balance).
- **Transactions**: List, Add, and Delete financial records.
- **State Management**: Using **Pinia** stores (`auth`, `finance`).

## Scripts
- `npm install`: Install dependencies.
- `npm run dev`: Start dev server.
- `npm run build`: Build for production.
